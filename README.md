# URL Shorty
Simple URL shortener made with Typescript, SQLite and Zod.

## Features
- Shortening URL
- CRUD functionalities
- Grouping
- Custom prefix

## Installation and deployment

#### Install
`npm run fresh:install`

#### Build

`npm run build`

#### Start

`npm run start`
